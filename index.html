Google Maps API Anahtarı

ADIM 1
API key nereden alınır?
  1. console.cloud.google.com adresinde bir proje oluştur.
  2. "APIs & Services > Library" içinden Directions API, Geocoding API ve Maps JavaScript API'yi etkinleştir.
  3. Faturalandırma (billing) hesabı bağlanmalı — Google her ay ücretsiz kullanım kredisi tanır, haftalık kullanım normalde bu kotayı aşmaz.
  4. "Credentials" bölümünden bir API key oluştur.
  5. Bu anahtar sadece tarayıcında saklanır (localStorage), hiçbir sunucuya gönderilmez.

Bu sayfayı bir yerel sunucudan aç (file:// değil) — API key kısıtlamalarıyla daha az sorun yaşarsın.
